ntec

  • tables
  • charts
  • monitorization objects
  • filters
  • free draw objects
  • sql analysis tool

Admin features:

  • backoffice menus
  • frontoffice config
  • options template config
  • page config
  • templates config
    • CSS
    • charts
    • monitorization objects
  • users config
